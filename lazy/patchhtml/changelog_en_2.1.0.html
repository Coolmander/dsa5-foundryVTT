Important New Features

Improved mundane Combat System

During the process of the implementation of the Compendium I the combat system has received a lot of improvements to faciliate automation:

Subcategories for Combat Special abilities

Combat Special Abilities are now separated into subcategories (f.e. special maneuever, passive). By default only one basic and special maneuver can now be selected in combat. This limitation can optionally be turned off in the configuration menu.

More Options for Combat Special Abilities

Combat Special Abilities can now override damage values, contain damage formulas and transfer active effects similar to spells on the target among other minor improvements.

Tracking of ongoing actions

The combat tracker can now show ongoing actions for combatants. The reloading action is the first element to be tracked there with preparation of spells and liturgies following in the near future.

Applications

Applications for skills can now be added on the character sheet. The compendium includes all base applications ready to be added to any actor sheet. (Using all available applications is not recommended for all actors for performance reasons).

Game Master Menu

The button menu on the left side now contains the game master menu. This menu aims to help the GM in certain situations and will be expanded in future patches. For now, you can do blind rolls for all players (f.e. perception), have access to chat command features via buttons and distribute adventure points. Also you will get a tiny overview over the characters' most important stats. This will also be the place for more advanced functions (e.g.familiars getting 25% AP). Feel free to add improvement proposals.

Enhancements

  • All skills now have new icons. Make sure to migrate your actors to see them!
  • Status effects are now hidden by default for players that don't own the actor with the effect. This can optionally be disabled in the configuration.
  • The soundconfig covers success and failures now too.
  • Improved whisper command a little.
  • Poisons and Diseases can now transfer Active Effects similar to spells.
  • Encumbrance can now be manually modified
  • Showing the artwork of actors (rightclick on the item sheet on the image) to players doesn't reveal the name anymore.
  • Property Knowledge and Property Focus should now automatically take effect.
  • All Armory Items now contain Active Effects where applicable
