2.2.0 Important new features (7/30/2021)

Library filtering upgrade

The library has new advanced filters. You can enable those with a click on the brain icon next to "search". Among many other options you can search for creatures by type or Special Abilities by subcategory.
While this is a huge improvement on the usability of the library, it is still an intermmediate state. The memory footprint is still rather high and needs further improvement. Also pagination might not be perfect yet.
WARNING: It seems Firefox is creating a memory leak using the library browser if you refresh the tab too often. If you are using Firefox you might want to close the tab and reopen a new one every now and then.
Chromium based browsers seem to work fine.

Journal Browser enhancements

The journal browser now allows searching for chapter titles in books. You can additionally enable a content search with a click on the "Full text search" button next to the search window.
Moreover, the journal browser now allows pinning journal entries. This helps a lot if you keep forgetting that one rule and have to search it again and again or whenever you need the information of multiple pages at once for an adventure (f.e. current scene and rumors tab).
To pin a journal entry open one in the browser and press the pin button or drag and drop any journal entry onto the journal browser. You can pin messages from any source. To remove pins rightclick their button in the journalbrowser.
The pinned messages can be different for each sl/player. So players can use this feature too to track important handouts or notes.

If you buy a service from a merchant, you won't get an item transferred to the buyer's inventory anymore. Instead a chat message stating the successfull trade will appear.   • The 'familiar' trait has been migrated to Active Effects.

  • The current target will now be shown on the attack dialog.
  • Paying and selling stuff to a merchant should play a small sound if you have game audio bundles installed.
